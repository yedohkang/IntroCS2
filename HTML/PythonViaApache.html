STEPS TO PRODUCE TEXT IN A BROWSER FROM A PYTHON FILE
STEP # STEP
0 Create a python file (.py)
1 View its read, write, execute rights using ls -1
2 Run the python file
3 Grant executing rights using chmod +x filename
4 To check and view, use ls -1
5 Run the file as a command (/pathtofile/filename)
6 Insert shbang and the path to python by using which python to figure out the path (ex. #! /usr/bin/python)
7 To check, run the file as a command using its /pathtofile/filename
8 Place the python file into your public html folder
9 Insert print 'content-type: text/html\n' after the shbang+usrbin but before the actual content of the file
10 Open the file in your browser to view results!


MISSTEPS
